# Сайт рассадки — Кирилл & Анна

Мобильный сайт для гостей свадьбы (10.07.2026). Гости сканируют QR-код и находят своё место за столом.

---

## Публикация на Netlify (рекомендуется)

1. Откройте [app.netlify.com/drop](https://app.netlify.com/drop)
2. Перетащите **всю папку** `seeting` в окно браузера
3. Дождитесь деплоя — получите ссылку вида `https://random-name.netlify.app`
4. Создайте QR-код на эту ссылку **без** `?admin=1`

### Какие файлы должны быть на хостинге

```
index.html
styles.css
config.js
data.js
script.js
admin.js
guests.json      ← главный список гостей
_headers         ← кэш и безопасность (Netlify)
netlify.toml
```

Файл `README.md` можно не загружать — на работу сайта не влияет.

---

## Перед публикацией — чеклист

- [ ] Сменить пароль админки: `ADMIN_PIN_HASH` в `config.js` (см. ниже)
- [ ] Заменить демо-имена в `guests.json` на реальный список (или через админку → экспорт)
- [ ] Проверить сайт локально через локальный сервер (см. ниже)
- [ ] Загрузить **все файлы** из списка выше
- [ ] Сохранить ссылку `?admin=1` только у себя
- [ ] Создать QR-код на основную ссылку (без `?admin=1`)

## Смена пароля админки

Пароль по умолчанию: `anna2026` — **обязательно смените перед публикацией**.

Сгенерируйте хеш нового пароля:

```bash
node -e "const c=require('crypto');console.log(c.createHash('sha256').update('ВАШ_ПАРОЛЬ').digest('hex'))"
```

Вставьте результат в `config.js` → `ADMIN_PIN_HASH`.

## Локальная проверка

Сайт загружает `guests.json` через сеть — при открытии `index.html` двойным кликом данные могут не подгрузиться.

```bash
# Python
python -m http.server 8080

# или Node.js
npx serve .
```

Откройте `http://localhost:8080`

## Админ-панель

### Вход
```
https://ваш-сайт.ru/?admin=1
```

### Возможности
- Добавление, редактирование, удаление гостей
- Назначение стола (1–5) и места (1–6)
- Сокращения для поиска (Катя → Екатерина)
- Экспорт / импорт `guests.json`

### Обновление рассадки для всех гостей

1. Внесите правки в админке (`?admin=1`)
2. Нажмите **«Экспорт JSON»** — скачается `guests.json`
3. Загрузите файл на Netlify **поверх старого** (Deploys → перетащить файл, или через панель)
4. Гости увидят изменения при следующем открытии сайта

---

## Как работают данные

| Источник | Когда используется |
|----------|-------------------|
| `guests.json` на хостинге | Для всех гостей (приоритет) |
| `GUESTS_LIST` в `config.js` | Запасной вариант, если json недоступен |
| localStorage браузера | Черновик правок в режиме `?admin=1` |

## Скорость загрузки

Сайт оптимизирован для мобильных и слабого интернета на площадке:

- **Без внешних шрифтов** — системные Georgia + system-ui
- **`guests.json` кэшируется** отдельно от кода
- **`admin.js` грузится только** по ссылке `?admin=1`
- **~85 КБ** суммарно при первом визите (≈25–35 КБ с gzip на Netlify)
